AI와 대화할 때 더 나은 결과를 얻고 싶으신가요? PromptLint는 프롬프트를 위한 품질 검사 도구입니다. Grammarly가 글쓰기를 돕듯이, PromptLint는 더 나은 AI 대화를 만들어줍니다.


■ 주요 기능

▸ 실시간 품질 분석
  • GOLDEN 6차원 평가 (목표, 출력, 제약, 데이터, 평가, 다음 단계)
  • A부터 F까지 등급으로 명확한 피드백
  • 구체적인 개선 제안과 예시

▸ 스마트한 개선 제안
  • 3단계 개선안 자동 생성 (보수적/균형/포괄적)
  • 선택적 AI 기반 고급 개선 (Claude API 연동)

▸ 개인화된 학습
  • 프롬프트 히스토리 추적
  • 약점 분석 및 맞춤형 팁 제공

▸ 편리한 워크플로우
  • 전역 단축키로 어디서나 실행 (⌘⇧P)
  • 클립보드에서 바로 분석
  • 시스템 트레이에서 빠른 접근


■ GOLDEN 평가 기준

프롬프트의 6가지 차원을 분석합니다:
  • Goal (목표) - 명확한 목적이 있는가?
  • Output (출력) - 원하는 결과 형식이 정의되었는가?
  • Limits (제약) - 적절한 범위와 제약이 설정되었는가?
  • Data (데이터) - 충분한 맥락이 제공되었는가?
  • Evaluation (평가) - 품질 기준이 있는가?
  • Next (다음 단계) - 후속 작업이 계획되었는가?


■ 이런 분들께 추천합니다

  • 개발자 - 코드 생성, 디버깅, 리뷰 프롬프트 개선
  • 기획자 - 기획 문서, 요구사항 정리
  • 마케터 - 카피라이팅, 콘텐츠 제작
  • 연구자 - 논문 분석, 데이터 해석
  • AI 입문자 - 효과적인 대화법 학습


■ 개인정보 보호

  • 모든 분석은 로컬에서 실행됩니다
  • 히스토리는 사용자 Mac에만 저장됩니다
  • AI 개선 기능은 선택사항입니다 (Claude API 필요)
  • 데이터 수집 및 추적 없음


■ 시스템 요구사항

  • macOS 12.0 (Monterey) 이상
  • Apple Silicon 또는 Intel Mac
  • 50MB 여유 공간
